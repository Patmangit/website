Building blocks

Entities

Long-lived objects defined by their identity.

Value objects

Immutable objects solely defined by their attribute values.

Aggregates

Sets of objects bound by a root entity.

Factories

Objects that create other domain objects.

Repositories

Objects responsible for storing and retrieving aggregates.

Services

Standalone logic within the context of the domain.

Domain events

Capture the memory of something interesting which affects the domain.

Interfaces

Finders

Objects retrieving data efficiently.

Pagination

Assisted slicing of big data sets.

Additional features

Audit

Automatically trace user actions.
