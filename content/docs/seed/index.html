Release notes

Reference

Core

Provides application lifecycle management, dependency injection, configuration and other common framework functionality.

Security

Allows code execution restriction based on a customizable and extensible security model.

Web

Supports servlet based Web applications and provide related services, like static resource serving.

REST

Supports JAX-RS based REST resources in a Web application.

Transactions

Enables usage of transactional resources from a SEED application.

Persistence

Supports various persistence paradigms, like relational databases, graphs, key-value, ...

JMS

Supports JMS 1.1 asynchronous queue or topic based communications.

Web Services

Allows to consume or produce JAX-WS based Web Services.

Validation

Supports JSR-303 and JSR-349 bean validation.

Spring

Bi-directional bridge between Spring application contexts and SEED managed injection.

Shell

Allows SSH interaction with a running SEED application.

Testing

Tools and helpers for various kind of testing.

Batch

Support for long-running batches in dedicated JVMs.

Scheduling

Support periodic task execution scheduling.

Cache

Support for applicative cache.

Mail

Support for receiving and sending mails.

Metrics

Support for monitoring application metrics.
