Core

Provides application initialization, resource loading, MVC engine, configuration and other common framework functionality.

User interface

Provides services and components to build general purpose rich user interfaces.

Data visualization

Provides various style of dynamic data visualization like charts.

Touch

Provides touch-based interactions and components to enable applications in kiosk mode.

Extras

Provides various add-ons for extending your application capabilities.
