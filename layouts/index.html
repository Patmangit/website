Enjoy building modular and scalable Web applications

SeedStack is a full-stack solution enabling teams to build great applications easily and efficiently. It includes the best Java/Web technologies and promotes an architecture that scales from one developer to vast organizations. It is simple for the newcomer yet powerful and extensible for the seasoned developer.

Write better code

Modular architecture, dependency injection, Java and Web standards, hypermedia APIs...
Opinionated choices that you can rely on.

Ship faster

Quick startup with project templates, Convention-over-Configuration, sensible defaults...
Ship a production-ready application in no-time.

Focus on business

Domain-Driven Design framework, no plumbing code, production-ready...
Leverage technologies with minimal cost.

Compose applications

Open/closed design, rich APIs and SPIs, automatic integration of modules, add-ons library...
Composability at every level of the stack.

Main components

SeedStack solution is built around three frameworks. A Java framework which provides simple and efficient access to useful technologies. A Business framework based on Domain-Driven Design principles for expressing your business. And a Web framework which enables to quickly develop ergonomic frontends.
